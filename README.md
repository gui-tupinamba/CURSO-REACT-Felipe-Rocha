# Projeto REACT com Vite
## Um simples gerenciador de task que salva as taks no Local Storage do navegador do usuário.
### Esse é o primeiro protótipo, depois eu farei algo mais avançado nesse repositório.

##Ferramentas utilizadas:
- [Ract Router](https://reactrouter.com/start/framework/routing) Utilizado para usar rotas diferentes para cada componente
- [UUID](https://www.npmjs.com/package/uuid) Utilizado para criar ID único para cada "Task"
- [Lucide](https://lucide.dev/guide/installation) Utilizada para os icons dentro da aplicação
- [Tailwind](https://tailwindcss.com/)
